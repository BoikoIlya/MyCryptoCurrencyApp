# MyCryptoCurrencyApp

Stack:
- Kotlin
- Clean archetecture
- MVVM
- Jetpack Compose
- Compose navigation
- Retrofit
- Hilt
- Coroutines
- Room

This app get list of cryptocurrencies, and shows details of them.
